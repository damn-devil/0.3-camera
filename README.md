# 0.3MP — Old Phone Camera

Симулятор камер старых телефонов (Nokia, Siemens, Sony Ericsson, Motorola, Sharp) начала 2000-х.

Всё обрабатывается локально на устройстве через Canvas — шум, banding, bloom, цветовые сдвиги, виньетка. Сервера нет, данные никуда не уходят.

## Как запустить на iPhone

1. Залей папку на GitHub Pages или любой HTTPS-хостинг
2. Открой сайт в Safari
3. Нажми Share → Add to Home Screen
4. Готово
